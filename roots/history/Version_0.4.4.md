# Version 0.4.4

该版本发布于2023年6月18日，大幅调整了网站的内容路径，重写了[内容总览](/roots/overview)页面，添加了面包屑导航，新增了[八年级 / 下册 / 八校联考优秀作文](/八年级/下册/八校联考优秀作文)页面，给[七年级 / 上册 / 期末考试优秀作文](七年级/上册/期末考试优秀作文)添加了按需加载图片的功能。

**更详细的更新记录请见：[GitHub上从v0.4.3到v0.4.4 Alpha 6的各文件详细对比](https://github.com/zz19z-2021-2/zz19z-2021-2.github.io/compare/8cf61e64751242693163072d98bbef6b2ce42b4c...3f02d43c32d7de2a625ba3d704f6377a5831d4a6)。**

## 新内容

- 新增“面包屑导航”，即现在的每个页面上会显示当前页面的URL位置。
    - 例如，现在[八年级 / 下册 / 八校联考优秀作文](/八年级/下册/八校联考优秀作文)（位于`/八年级/下册/八校联考优秀作文`）的页面顶部会显示“首页 > 八年级 > 下册 > 八校联考优秀作文”，以便读者区分当前页面所属的年级、上下册，以及获取当前页面在网站的位置等信息。
- 新增[网站代码备份](/roots/backups)页面，用于存储未来可能需要重复使用的代码。
- 新增[八年级 / 下册 / 八校联考优秀作文](/八年级/下册/八校联考优秀作文)页面。
    - 因为页面图片较多，本页面使用了按需加载图片的方式。

### 技术性

- 添加了[八年级 / 下册 / 八校联考优秀作文](/八年级/下册/八校联考优秀作文)的图片文件，存放于`/Files/八年级/下册/八校联考优秀作文/`。

## 更改

- 修改所有普通页面名称的格式为`/年级/上下册/页面标题`。
    - 例如，“八年级上册｜期末背书规划”现在称作[八年级 / 上册 / 期末背书规划](/八年级/上册/期末背书规划)。
- 同时删除所有页面的标题前的年级和上下册前缀。
    - 例如，[八年级 / 上册 / 期末背书规划](/八年级/上册/期末背书规划)页面的标题现在写作“期末背书规划”。
- 现在，空页面的页面头部横幅上会以“笑草班”为标题了。
- 现在[首页](/)的“经典诵读”板块中，词的上下片会有更清晰的段落提示了。
- 现在每个段落前都会有两个汉字的缩进，使网站段落更鲜明。
- 现在如果某个段落需要跨行显示，占满一整行的每个英语单词或中文汉字间会拉开一定距离，以保证段落两端的文字对齐。
- 几乎重写[内容总览](/roots/overview)页面。
    - 现在通往根页面的按钮在屏幕横向宽度不足时，会自动切换为全宽按钮。
    - 现在通往普通页面的按钮会以年级分类在对应的列表组中，在屏幕横向宽度不足时，会以一整个列表组为单位分栏，并将按钮切换为全宽。
- 所有带目录的页面现在都可以自动调整目录分栏，以便充分利用屏幕空间。
- 由于网站导航栏更改，将[八年级 / 上册 / 政治学习资料整理](/八年级/上册/政治学习资料整理)页面中，切换口诀和完整内容的按钮从胶囊修改为标签页。
- 现在[个人空间](/其他/个人空间)页面的姓名输入框和按钮会在屏幕横向宽度不足时换行显示。
- 现在[七年级 / 上册 / 期末考试优秀作文](七年级/上册/期末考试优秀作文)也有按需加载图片的功能了。

### 技术性

- 将`/Files/`下的所有文件移至新文件夹，以匹配根目录下`.md`文件的路径。
    - 将`/Files/笑草说/`移到了`/Files/七年级/笑草说/`。
- 将所有普通页面文件按`/年级/上下册/页面名`移到对应文件夹下，以匹配页面命名规范。
    - 例如，[八年级 / 上册 / 期末背书规划](/八年级/上册/期末背书规划)现在位于`/八年级/上册/期末背书规划`。
    - 如果内容不符合任何年级，则会被移到`/其他/`。
        - 例如，[资料整理](/其他/资料整理)位于`/其他/资料整理`。
    - 如果内容不符合某年级的上/下册，则会被直接放在`/年级/`。
        - 例如，[七年级 / 笑草说](七年级/笑草说)位于`七年级/笑草说`。
- 将所有页面根部文件（包括[帮助 / 反馈](roots/feedback)页面、[内容总览](/roots/overview)页面、[更新历史](/roots/history)页面、[网页备份](/roots/backups)页面）移到`/roots/`，减少未来可能的大量页面索引。
- 将每个人的[个人空间](/其他/个人空间)移至`/其他/Spaces/`。
- 现在，空的[个人空间](/其他/个人空间)页面会由GitHub以`_layouts/space.html`为模版自动生成了。
- 现在，所有页面的样式会在`/_includes/head-custom.html`引用自外部样式表了。
- 简化本页面自动生成目录的JavaScript代码。
- 因为网站整体段落样式更改，所以删除了部分页面中多余的`<p>`元素，使页面维护更方便（GitHub会自动生成`<p>`段落）。

## 修复

- 浏览器工具栏中的标题现在是“笑草班”（修复前为`zz19z-2021-2.github.io`）。
- [内容总览](/roots/overview)页面现在按钮和标题与分割线之间不再有无意义的空白。
- 隐藏页面[你好](/niganma)现在也会有相同的页面头部了。
